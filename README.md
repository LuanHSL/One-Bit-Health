# One Bit Health

> Requisitos mínimos
- [NodeJS](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com)
- [Expo](https://docs.expo.dev/)

## Instalação

```sh
git clone git@github.com:LuanHSL/One-Bit-Health.git
```

```sh
cd One-Bit-Health
```

- Instalar as dependências

```sh
yarn install
```

- Subir o servidor

```sh
yarn start
```
